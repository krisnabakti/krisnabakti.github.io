Smartponics is an ESP32-based IoT device to monitor and control  farming parameters such as nutrient level, pH value, water & air temperature, and sunlight intensity in real-time.

### Features
- AWS MQTT protocol for real-time response
- Custom-built PCB for secure electrical connection between modules
- RTC-based power control circuit for periodical logging
- In-house design of EC sensor for stability over wide range of EC value
- Google Sheets as UI for logging, monitoring, and controlling farming parameters
